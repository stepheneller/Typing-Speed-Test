** Typing Speed Test **

Basic Python app for testing your typing speed.

Make sure all files are located in the same directory before starting the main.py in an IDE or terminal.

Once you start the program, you can press the start button to begin the test.
You will have 60 seconds to type the phrase seen on the screen.
After the time is up your WPM and accuracy will be displayed on-screen.
Press the rest button to restart the program and try again.

Package requirements:
 - tkinter
 - math
